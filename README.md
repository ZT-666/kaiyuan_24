大学生智慧闹钟

介绍
免费且易于帮助大学生起床去早八，它致力于提醒课程时间给大学生，让体验者能够便于学习

背景

在如今大学生活中，越来越多大学生因忙于致力发展各项能力，而忘却课程时刻，因此设计一个大学生智慧闹钟提醒帮助大学生课程时间
